# README

個人的な思考や見解を置いておく場所です。

このサイトは [ZEN](https://zccz14.github.io/ZEN) を使って構築しました。

ZENは純粋にVibe-Codingによって生まれたプロジェクトで、私はコードを一行も書かずに我慢し、全工程で「現場監督」のような役割に徹しました。この姿勢は今後も貫いていきます。

使い方はシンプルで、個人的な審美眼や好みをいくらか反映しています。ただし、現時点ではまだ正式リリースされておらず、私の期待にはまだ少し距離があります。
