现在是 2026年2月2日，周一，凌晨。

解决了 Claude Opus 模型的 Quota 用尽的问题。找了个中转的服务，成本甚至低过 DeepSeek。感受到了 AI 公司的毛利率有多么可怕。

Anthropic [官方报价](https://platform.claude.com/docs/zh-CN/about-claude/pricing)，按 1M Tokens 计算：

| Model           | Base Input Tokens | 5m Cache Writes | 1h Cache Writes | Cache Hits & Refreshes | Output Tokens |
| --------------- | :---------------: | :-------------: | :-------------: | :--------------------: | :-----------: |
| Claude Opus 4.5 |        $5         |      $6.25      |       $10       |         $0.50          |      $25      |

而中转服务商报价：输入 0.1890 CNY，输出 0.9470 CNY，同样是 1M Tokens。

以汇率 1 USD = 7.0 CNY 计算，官方的 INPUT 成本是 35 CNY，而中转服务商的 INPUT 成本是 0.189 CNY，这有 **185.2 倍**的差距；官方的 OUTPUT 成本是 175 CNY，而中转服务商的 OUTPUT 成本是 0.947 CNY，这有 **184.8 倍**的差距。总结下来，Anthropic 官方的报价大约是中转服务商报价的 185 倍。

而显然，Anthropic 的真实成本一定比中转服务商还要低。即便以中转服务商的价格计算，Anthropic 的毛利率也高达 99.46%，中转服务商显然也需要赚取一定利润，那么 Anthropic 的毛利率就更高了。而且，Anthropic 显然也不是以成本价为中转服务商供货的。细思极恐。

本以为这个世界的 AI 很贵，就连人民的 AI (DeepSeek，性价比之王) 也得卖 2 CNY INPUT + 3 CNY OUTPUT。但一个中转服务商都能把 SOTA 模型(Claude Opus)的成本压到 DeepSeek 的 1/3 以内。这个世界还是太疯狂了。

好吧，总体来说，中转服务器仍然有较高的隐私风险，但是针对很多非敏感任务（例如做开源项目、学习研究等），使用中转服务器仍然是一个不错的选择。毕竟成本差距实在太大了。
