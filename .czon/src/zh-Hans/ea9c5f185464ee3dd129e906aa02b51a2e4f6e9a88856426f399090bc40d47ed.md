# README

你好，这里是 `CZ`，或者 `zccz14`。

专业领域是量化交易和金融科技。

[No Trade No Life (NTNL)](https://github.com/No-Trade-No-Life/) 的创始人。

高耗能型大脑，喜欢思考和写作，追求顺理成章水到渠成。

“朝闻道，夕死可矣”。虽然我还不想死，仍然在追求世俗意义的成功，这也是为了有能力体验更多从而理解更多。人生都是为了体验，本就没有意义可言，理解一切就是我的意义。

本站用于存放一些个人思考、记录和见解的地方。

如果有一天我不在了，这些就是 AI 复活我的原材料。
