# 資本市場の三体力学仮説

2026-02-07

## 核心思想

資本市場は、本質的に異なる3種類の資本から構成される**三体システム**であると仮定します。それらは相互作用を通じて、ボラティリティ・クラスタリング、市場の崩壊と回復サイクルなどの創発現象を含む、複雑な力学的振る舞いを生み出します。

天体力学における三体問題と同様に、このシステムには**定常解が存在しない可能性**があり、リミットサイクル、準周期的、またはカオス的な振る舞いを示すと考えられます。

## 三体の定義

市場参加者の本質的な違いは、その属性（個人投資家、機関投資家、マーケットメイカー）ではなく、**価格変化に対するフィードバックの性質**にあります。

### モメンタム資本 M (Momentum Capital)

**定義**：価格変化に対して正のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} > 0$$

**行動特性**：

- 上昇時に買い、下落時に売る（順張り）
- レバレッジを用いて利益/損失を増幅
- 短期保有の傾向
- モメンタム取引、トレンドフォロー

**システムへの影響**：

- 価格変動を増幅
- 不安定化する力
- トレンドと崩壊を生み出す

**典型的な代表**：投機家、トレンドトレーダー、高レバレッジ取引者、受動的な損切りを行う保有者

### バリュー資本 V (Value Capital)

**定義**：価格変化に対して負のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} < 0 \quad (\text{ただし } S < S^*)$$

ここで、$S^*$ は当該資本が評価する内在価値です。

**行動特性**：

- 安値で買い、高値で売る（逆張り）
- 価値判断に基づく取引
- 長期保有の傾向
- 平均回帰、バリュー投資

**システムへの影響**：

- 価格変動を抑制
- 安定化する力（能動的）
- 市場の下支えと抵抗を提供

**典型的な代表**：バリュー投資家、逆張り投資家、アービトラージャー

### 流動性資本 L (Liquidity Capital)

**定義**：価格変化に対して方向性のない反応を示す流動性資本。

$$\frac{d(\text{ポジション})}{dS} \approx 0$$

**行動特性**：

- 両建ての価格提示、スプレッドの獲得
- 方向性のあるエクスポージャーを持たない（または迅速にヘッジ）
- 継続的に売買流動性を提供
- リスク許容度は限定的

**システムへの影響**：

- 取引コストを低下
- 安定化する力（受動的）
- 価格ショックを緩衝

**典型的な代表**：マーケットメイカー、流動性提供者、高頻度取引業者（マーケットメイキング型）

## 三体の相互作用

### 作用行列

| 作用側 → 被作用側 | モメンタム資本 M   | バリュー資本 V   | 流動性資本 L |
| ----------------- | ------------------ | ---------------- | ------------ |
| **モメンタム資本 M** | —                  | 買い機会を創出   | 流動性を消耗 |
| **バリュー資本 V**   | 極端な行動を抑制   | —                | 信頼を回復   |
| **流動性資本 L**     | ショック効果を制約 | 取引の便宜を提供 | —            |

### 詳細なメカニズム

**M → L：流動性の消耗**

モメンタム資本の順張り行動は大量の一方向注文フローを生み出し、マーケットメイカーの在庫を消耗させ、より大きなリスクエクスポージャーを負わせます。ボラティリティが高すぎる場合、マーケットメイカーは撤退を選択します。

**L → M：ショック効果の制約**

十分な流動性は、モメンタム資本による価格への衝撃を緩和します。市場の流動性が深い場合、高レバレッジ取引であっても価格を大きく動かすことは困難です。流動性は「ショックアブソーバー」です。

**M → V：買い機会の創出**

モメンタム資本のパニック売りは価格を内在価値以下に押し下げ、バリュー資本にとっての「価値の窪地」を創出します。これはモメンタム資本からバリュー資本への**機会の移転**です。

**V → M：極端な行動の抑制**

バリュー資本の逆張り操作は価格の下支えを提供し、価格が一方的に下落する確率を下げ、順張り戦略の期待収益を低下させます。合理的なモメンタム資本はこれにより規模を縮小します。

**V → L：信頼の回復**

バリュー資本の参入は「市場には底がある」というシグナルを発し、マーケットメイカーの極端な損失に対する懸念を軽減し、流動性の回帰を促します。バリュー資本はマーケットメイカーの「保険」です。

**L → V：取引の便宜の提供**

十分な流動性により、バリュー資本は低コストでポジションを構築でき、大口注文でも過度のスリッページを生じず、バリュー資本の資金効率を高めます。

## フィードバック・ループ

### 正のフィードバック・ループ（不安定）

$$M \uparrow \to \sigma \uparrow \to L \downarrow \to \text{価格ショック} \uparrow \to \sigma \uparrow \to M \uparrow \text{ (または強制決済)}$$

モメンタム資本の増加 → ボラティリティ上昇 → 流動性の撤退 → 価格ショックの激化 → ボラティリティのさらなる上昇 → モメンタム資本の継続的増加または強制決済

これは**崩壊スパイラル**のメカニズムです。

### 負のフィードバック・ループ（安定）

$$\sigma \uparrow \to S \downarrow \to V \uparrow \to L \uparrow \to \sigma \downarrow$$

ボラティリティ上昇 → 価格下落 → バリュー資本の参入 → 流動性の回復 → ボラティリティ低下

これは**回復メカニズム**です。

### もう一つの負のフィードバック・ループ

$$V \uparrow \to M \downarrow \to \sigma \downarrow \to L \uparrow$$

バリュー資本の増加 → モメンタム資本の縮小 → ボラティリティ低下 → 流動性の回復

## システムの相転移

システムの状態は、**どのループが支配的か**によって決まります：

- **正のフィードバック > 負のフィードバック**：システムは崩壊に向かう
- **負のフィードバック > 正のフィードバック**：システムは安定に向かう
- **臨界点**：システムは相転移の境界にある

### 安定性指標

無次元の**市場緊張度**を定義します：

$$\Theta = \frac{M \cdot \sigma}{L \cdot (1 + V)}$$

- $\Theta \ll 1$：負のフィードバックが支配的、市場は安定
- $\Theta \gg 1$：正のフィードバックが支配的、市場は崩壊に向かう
- $\Theta \approx 1$：臨界状態

注意：この指標はシステム状態の**創発的結果**であり、事前に設定された閾値ではありません。システムは正負のフィードバックの競合の中で自発的に相転移を生み出します。

## 三体の生態的地位

| 資本タイプ     | 生態的役割     | システム安定性への影響 | 収益源          |
| -------------- | -------------- | ---------------------- | --------------- |
| モメンタム資本 M | エネルギー注入者 | 不安定化               | ボラティリティ × 方向性判断 |
| 流動性資本 L   | 緩衝材         | 安定化（受動的）       | 売買スプレッド  |
| バリュー資本 V | 負のフィードバック制御器 | 安定化（能動的）       | 価値回帰        |

**生態的バランス**：健全な市場には三者が共存する必要があります。

- モメンタム資本が欠如：市場は停滞し、変動も取引機会もない
- 流動性資本が欠如：取引コストが高く、市場効率性が低い
- バリュー資本が欠如：市場は脆弱で、崩壊しやすく、ゼロに帰する可能性すらある

## 従来の分類との関係

| 従来の分類     | 本質的帰属     | 説明                 |
| -------------- | -------------- | -------------------- |
| 投機家         | モメンタム資本 M | 順張り、レバレッジ増幅 |
| 投資家         | バリュー資本 V   | 逆張り、価値判断     |
| マーケットメイカー | 流動性資本 L | 両建て価格提示、スプレッド獲得 |
| トレンドトレーダー | モメンタム資本 M | モメンタム戦略       |
| アービトラージャー | バリュー資本 V | 価格差の収斂         |
| パッシブ・インデックスファンド | ほぼ L       | リバランス時に弱い負のフィードバックを生む |

注意：同一の参加者でも、時と場合によって異なる役割を果たす可能性があります。分類の本質は**行動パターン**であり、属性ラベルではありません。

## 三体の類推

市場の三体は、天体力学における三体問題と深い類似性を持ちます。

**類推の鍵**：三体問題の本質は、**質量が近い3つの天体が相互作用する**ことです。互角であるがゆえに、いずれの天体もシステムを支配できず、カオス的な振る舞いが生じます。

市場においても、M、V、L の3種類の資本は同様に**互角**です：

- M が V と L を大きく上回る場合：市場は一方的に暴騰暴落した後、ゼロに帰する（バブル崩壊）
- V が M と L を大きく上回る場合：市場はほとんど変動しない（停滞）
- L が M と V を大きく上回る場合：価格は完全に外部情報によって決定される（完全効率的市場）

三者が互角である時のみ、市場は真の複雑なダイナミクスを示します。

**天体三体からの教訓**：

- 二体問題には解析解がある（楕円軌道）
- 三体問題には一般に解析解がなく、初期値に敏感
- 軌道は周期的、準周期的、またはカオス的になりうる

**市場三体からの推論**：

- **長期予測は不可能**：システムは初期値に敏感
- **短期的特性は予測可能**：トレンドとボラティリティ・クラスタリングは短期的現象
- **統計的規則性は頑健**：収益率分布のファットテール、ボラティリティ・クラスタリングなどの巨視的統計特性は安定している
