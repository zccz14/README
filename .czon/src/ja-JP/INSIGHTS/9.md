# 資本市場の三体力学仮説

2026-02-07

**なぜ市場の行動には法則性があると同時に不確実性が満ちているのか？**

- なぜ市場には常に機会があるが、不確実なのか？
- なぜ市場は常に繰り返すが、再現しないのか？
- なぜ市場は予測可能な時と不可能な時があるのか？
- なぜ資本市場にはボラティリティ・クラスタリング、市場崩壊と回復の現象があるのか？
- なぜ市場には急変現象があるのか？
- なぜ市場には急騰後の急落というパターンがあるのか？
- なぜ市場にはゼロに帰するリスクがあるのか？
- なぜ市場の統計的特性（収益率分布のファットテール、ボラティリティ・クラスタリングなど）は頑健なのか？

## 核心思想

資本市場は**三体システム**であり、本質的に異なる三種類の資本で構成されていると仮定する。それらは相互作用を通じて、ボラティリティ・クラスタリング、市場崩壊と回復のサイクルなどの創発現象を含む、複雑な力学的行動を生み出す。

天体力学における三体問題と同様に、このシステムには**定常解が存在しない可能性**があり、代わりにリミットサイクル、準周期的、またはカオス的な振る舞いを示す。

## 三体の定義

市場参加者の本質的な違いは、その属性（個人投資家、機関投資家、マーケットメイカー）ではなく、**価格変化に対するフィードバックの性質**にある。

### モメンタム資本 M (Momentum Capital)

**定義**：価格変化に対して正のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} > 0$$

**行動特性**：

- 上昇時に買い、下落時に売る（上げ相場で買い、下げ相場で売る）
- レバレッジを用いて利益/損失を増幅させる
- 短期保有の傾向がある
- モメンタム取引、トレンドフォロー

**システムへの影響**：

- 価格変動を増幅させる
- 不安定化する力
- トレンドと崩壊を生み出す

**典型的な代表**：投機家、トレンドトレーダー、高レバレッジトレーダー、受動的ストップロスを持つ保有者

### バリュー資本 V (Value Capital)

**定義**：価格変化に対して負のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} < 0 \quad (\text{ただし } S < S^*)$$

ここで $S^*$ は、その資本が評価する内在的価値である。

**行動特性**：

- 安く買い、高く売る（逆張り操作）
- 価値判断に基づいて取引を行う
- 長期保有の傾向がある
- 平均回帰、バリュー投資

**システムへの影響**：

- 価格変動を抑制する
- 安定化する力（能動的）
- 市場の下支えと抵抗を提供する

**典型的な代表**：バリュー投資家、逆張り投資家、アービトラージャー

### 流動性資本 L (Liquidity Capital)

**定義**：価格変化に対して方向性のない反応を示す流動性資本。

$$\frac{d(\text{ポジション})}{dS} \approx 0$$

**行動特性**：

- 両建ての価格提示（ビッドとアスク）、スプレッドを稼ぐ
- 方向性のあるエクスポージャーを持たない（または迅速にヘッジする）
- 継続的に売買流動性を提供する
- リスク許容度は限定的

**システムへの影響**：

- 取引コストを低下させる
- 安定化する力（受動的）
- 価格ショックを緩衝する

**典型的な代表**：マーケットメイカー、流動性提供者、高頻度取引業者（マーケットメイキングタイプ）

## 三体の相互作用

### 作用行列

| 作用側 → 被作用側 | モメンタム資本 M   | バリュー資本 V   | 流動性資本 L |
| ----------------- | ------------------ | ---------------- | ------------ |
| **モメンタム資本 M** | —                  | 買い機会を創出   | 流動性を消耗 |
| **バリュー資本 V**   | 極端な行動を抑制   | —                | 信頼を回復   |
| **流動性資本 L**     | 衝撃効果を制約     | 取引の便宜を提供 | —            |

### 詳細なメカニズム

**M → L：流動性の消耗**

モメンタム資本の上げ相場買い・下げ相場売り行動は、大量の一方向のオーダーフローを生み出し、マーケットメイカーの在庫を消耗させ、マーケットメイカーに大きなリスクエクスポージャーを負わせることを強いる。ボラティリティが高すぎる場合、マーケットメイカーは撤退を選択する。

**L → M：衝撃効果の制約**

十分な流動性は、モメンタム資本による価格への衝撃を緩和する。市場の流動性が深い場合、高レバレッジ取引であっても価格を大きく動かすことは難しい。流動性は「ショックアブソーバー」である。

**M → V：買い機会の創出**

モメンタム資本のパニック売りは、価格を内在的価値以下に押し下げ、バリュー資本に「価値の窪地」を創出する。これはモメンタム資本からバリュー資本への**機会の移転**である。

**V → M：極端な行動の抑制**

バリュー資本の逆張り操作は価格の下支えを提供し、価格が一方的に下落する確率を低下させ、上げ相場買い・下げ相場売り戦略の期待収益を減少させる。合理的なモメンタム資本は、これにより規模を縮小する。

**V → L：信頼の回復**

バリュー資本の介入は「市場には底がある」というシグナルを発し、マーケットメイカーの極端な損失に対する懸念を軽減し、流動性の回帰を促す。バリュー資本はマーケットメイカーの「保険」である。

**L → V：取引の便宜の提供**

十分な流動性により、バリュー資本は低コストでポジションを構築でき、大口注文でも過度のスリッページを生じず、バリュー資本の資金効率を高める。

## フィードバックループ

### 正のフィードバックループ（不安定）

$$M \uparrow \to \sigma \uparrow \to L \downarrow \to \text{価格衝撃} \uparrow \to \sigma \uparrow \to M \uparrow \text{ (または強制決済)}$$

モメンタム資本の増加 → ボラティリティ上昇 → 流動性の撤退 → 価格衝撃の激化 → ボラティリティのさらなる上昇 → モメンタム資本の継続的増加または強制決済

これは**崩壊スパイラル**のメカニズムである。

### 負のフィードバックループ（安定）

$$\sigma \uparrow \to S \downarrow \to V \uparrow \to L \uparrow \to \sigma \downarrow$$

ボラティリティ上昇 → 価格下落 → バリュー資本の介入 → 流動性の回復 → ボラティリティ低下

これは**回復メカニズム**である。

### もう一つの負のフィードバックループ

$$V \uparrow \to M \downarrow \to \sigma \downarrow \to L \uparrow$$

バリュー資本の増加 → モメンタム資本の縮小 → ボラティリティ低下 → 流動性の回復

## システムの相転移

システムの状態は、**どのループが支配的か**によって決まる：

- **正のフィードバック > 負のフィードバック**：システムは崩壊に向かう
- **負のフィードバック > 正のフィードバック**：システムは安定に向かう
- **臨界点**：システムは相転移の境界にある

### 安定性指標

無次元の**市場緊張度**を定義する：

$$\Theta = \frac{M \cdot \sigma}{L \cdot (1 + V)}$$

- $\Theta \ll 1$：負のフィードバックが支配的、市場は安定
- $\Theta \gg 1$：正のフィードバックが支配的、市場は崩壊に向かう
- $\Theta \approx 1$：臨界状態

注意：この指標はシステム状態の**創発的結果**であり、事前に設定された閾値ではない。システムは正負のフィードバックの競合の中で自発的に相転移を生み出す。

## 三体の生態的地位

| 資本タイプ     | 生態的役割     | システム安定性への影響 | 収益源                |
| -------------- | -------------- | ---------------------- | --------------------- |
| モメンタム資本 M   | エネルギー注入者 | 不安定化               | ボラティリティ × 方向判断 |
| 流動性資本 L   | 緩衝材         | 安定化（受動的）       | 売買スプレッド        |
| バリュー資本 V   | 負のフィードバック制御器 | 安定化（能動的）       | 価値回帰              |

**生態的バランス**：健全な市場には三者共存が必要である。

- モメンタム資本が欠如：市場は停滞し、変動も取引機会もない
- 流動性資本が欠如：取引コストが高く、市場効率性が低い
- バリュー資本が欠如：市場は脆弱で、崩壊しやすく、ゼロに帰するリスクさえある

## 従来の分類との関係

| 従来の分類     | 本質的帰属     | 説明                     |
| -------------- | -------------- | ------------------------ |
| 投機家         | モメンタム資本 M | 上げ相場買い・下げ相場売り、レバレッジ増幅 |
| 投資家         | バリュー資本 V   | 安く買い高く売り、価値判断 |
| マーケットメイカー | 流動性資本 L   | 両建て価格提示、スプレッド稼ぎ |
| トレンドトレーダー | モメンタム資本 M | モメンタム戦略           |
| アービトラージャー | バリュー資本 V   | スプレッド収斂           |
| パッシブ・インデックスファンド | ほぼ L       | リバランス時に弱い負のフィードバックを生む |

注意：同一の参加者でも、時と場合によって異なる役割を演じる可能性がある。分類の本質は**行動パターン**であり、属性ラベルではない。

## 三体の類推

市場の三体は、天体力学における三体問題と深い類似性を持つ。

**類推の鍵**：三体問題の本質は、**質量が近い三つの天体が相互作用する**ことである。互角であるがゆえに、いずれの天体もシステムを支配できず、カオス的な振る舞いが生じる。

市場においても、M、V、L 三種類の資本は同様に**互角**である：

- M が V と L をはるかに上回る場合：市場は一方的に急騰・急落した後、ゼロに帰する（バブル崩壊）
- V が M と L をはるかに上回る場合：市場はほとんど変動しない（停滞）
- L が M と V をはるかに上回る場合：価格は完全に外部情報によって決定される（完全効率的市場）

三者が互角である時にのみ、市場は真の複雑なダイナミクスを示す。

**天体三体からの教訓**：

- 二体問題には解析解がある（楕円軌道）
- 三体問題には一般に解析解がなく、初期値に敏感である
- 軌道は周期的、準周期的、またはカオス的でありうる

**市場三体からの推論**：

- **長期予測は不可能**：システムは初期値に敏感である
- **短期的特性は予測可能**：トレンドとボラティリティ・クラスタリングは短期的現象である
- **統計的規則性は頑健**：収益率分布のファットテール、ボラティリティ・クラスタリングなどの巨視的統計的特性は安定している

## 補足：資本規模の長期的進化

三体モデルの核心は M-V-L 間の相互作用であるが、前提となる問題がある：**なぜ三種類の資本は長期的に共存し、互角であり続けるのか？**

これは、**収益率駆動型自然選択**（Return-Driven Selection）という従属メカニズムに依存している。

### メカニズムの説明

同一の資本タイプ内において、個々の収益率は分布を示す。個体の行動（拡大、縮小、退出）はその収益率と強く相関する：

- 高収益の個体は存続または拡大する傾向がある
- 低収益の個体は縮小または退出する傾向がある

多数の個体の統計的効果により、この傾向性はその資本タイプの総量の変化として現れる。

### 自発的調整

ある資本タイプが過剰な場合：

1. 内部競争が激化する
2. 平均収益率が低下する
3. 限界的な個体の退出が増加する
4. その資本タイプの総量が縮小する

ある資本タイプが不足している場合：

1. 内部競争が弱まる
2. 平均収益率が上昇する
3. 新たな資本の流入を引き寄せる
4. その資本タイプの総量が拡大する

### 各資本タイプの収益源と競争

| 資本タイプ | 収益源               | 過剰時の内輪もめ（内巻き）の現れ方             |
| ---------- | -------------------- | ---------------------------------------------- |
| M          | ボラティリティ、トレンド継続 | 互いに踏み合い、スリッページ増大、トレンドが前倒しされる |
| V          | 価値乖離、平均回帰   | 価値の窪地が奪い合われ、安全マージンが消失する |
| L          | 取引量、売買スプレッド | スプレッドが狭まり、マーケットメイキング利益が薄まる |

### 理論的位置付け

収益率駆動型自然選択は**遅い変数**のメカニズム（週から年の時間スケール）であり、三体相互作用は**速い変数**のメカニズム（秒から日の時間スケール）である。

この従属メカニズムは、三体システムの**存在性**と**持続性**——なぜ市場がある一つの資本タイプが独り勝ちする状態に進化しないのか——を説明するが、三体相互作用の核心的なダイナミクスを変えるものではない。

## 研究方向

1.  **相空間構造**：アトラクター、リペラー、分離曲面
2.  **時間スケール分離**：速い変数（価格）、遅い変数（資本構造）
3.  **統計的特性**：エルゴード性、不変測度、滞在時間分布
4.  **力学方程式**：この枠組みに基づく SDE システム（別稿で詳述）
