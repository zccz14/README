# 資本市場の三体力学仮説

2026-02-07

**なぜ市場の行動には法則性と不確実性が共存するのか？**

- なぜ市場には常に機会があるが、不確実なのか？
- なぜ市場は常に繰り返すが、再現しないのか？
- なぜ市場は予測可能な時と不可能な時があるのか？
- なぜ資本市場にはボラティリティ・クラスタリング、市場崩壊と回復の現象があるのか？
- なぜ市場には急変現象があるのか？
- なぜ市場には急騰後の急落というパターンがあるのか？
- なぜ市場にはゼロに帰するリスクがあるのか？
- なぜ市場の統計的特性（収益率分布のファットテール、ボラティリティ・クラスタリングなど）は頑健なのか？

## 核心思想

資本市場は**三体システム**であり、本質的に異なる三種類の資本で構成されていると仮定します。それらは相互作用を通じて、ボラティリティ・クラスタリング、市場崩壊と回復のサイクルなどの創発現象を含む複雑な力学的行動を生み出します。

天体力学における三体問題と同様に、このシステムには**定常解が存在しない可能性**があり、リミットサイクル、準周期的、またはカオス的な振る舞いを示します。

## 三体の定義

市場参加者の本質的な違いは、その属性（個人投資家、機関投資家、マーケットメイカー）ではなく、**価格変化に対するフィードバックの性質**にあります。

### モメンタム資本 M (Momentum Capital)

**定義**：価格変化に対して正のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} > 0$$

**行動特性**：

- 上昇時に買い、下落時に売る（順張り）
- レバレッジを用いて利益/損失を増幅
- 短期保有の傾向
- モメンタム取引、トレンドフォロー

**システムへの影響**：

- 価格変動の増幅
- 不安定化する力
- トレンドと崩壊の創出

**典型的な代表**：投機家、トレンドトレーダー、高レバレッジトレーダー、受動的ストップロス保有者

### バリュー資本 V (Value Capital)

**定義**：価格変化に対して負のフィードバックを生み出す取引資本。

$$\frac{d(\text{ポジション})}{dS} < 0$$

すなわち：価格上昇時にポジションを減らし、価格下落時にポジションを増やす。

通常、内在価値 $S^*$ がアンカーとして存在します：
- $S < S^*$ の時、買い傾向
- $S > S^*$ の時、売り傾向

**行動特性**：

- 安値で買い、高値で売る（逆張り）
- 価値判断に基づく取引
- 長期保有の傾向
- 平均回帰、バリュー投資

**システムへの影響**：

- 価格変動の抑制
- 安定化する力（能動的）
- 市場の下支えと抵抗の提供

**典型的な代表**：バリュー投資家、逆張り投資家、アービトラージャー

### 流動性資本 L (Liquidity Capital)

**定義**：価格変化に対して方向性のない反応を示す流動性資本。

$$\frac{d(\text{ポジション})}{dS} \approx 0$$

**行動特性**：

- 両建ての価格提示、スプレッドの獲得
- 方向性のあるエクスポージャーを持たない（または迅速にヘッジ）
- 継続的な売買流動性の提供
- リスク許容度は限定的

**システムへの影響**：

- 取引コストの低減
- 安定化する力（受動的）
- 価格ショックの緩衝

**典型的な代表**：マーケットメイカー、流動性提供者、高頻度取引業者（マーケットメイキング型）

## 三体の相互作用

### 作用行列

| 作用側 → 被作用側 | モメンタム資本 M   | バリュー資本 V   | 流動性資本 L |
| ----------------- | ------------------ | ---------------- | ------------ |
| **モメンタム資本 M** | —                  | 取引機会の創出   | 流動性の消耗 |
| **バリュー資本 V**   | 極端な行動の抑制   | —                | 信頼の回復   |
| **流動性資本 L**     | 衝撃効果の制約     | 取引の便宜提供   | —            |

### 詳細なメカニズム

**M → L：流動性の消耗**

モメンタム資本の順張り行動は大量の一方向のオーダーフローを生み出し、マーケットメイカーの在庫を消耗させ、より大きなリスクエクスポージャーを負わせます。ボラティリティが高すぎる場合、マーケットメイカーは撤退を選択します。

**L → M：衝撃効果の制約**

十分な流動性は、モメンタム資本による価格への衝撃を緩和します。市場の流動性が深い場合、高レバレッジ取引でも価格を動かすことは困難です。流動性は「ショックアブソーバー」です。

**M → V：取引機会の創出**

モメンタム資本の順張り行動は価格を内在価値から押し離し、バリュー資本に取引機会を創出します：

- パニック売り → 価格が内在価値を下回る → V の**買い**機会
- 熱狂的な買い → 価格が内在価値を上回る → V の**売り**機会

これはモメンタム資本からバリュー資本への**機会の移転**です。

**V → M：極端な行動の抑制**

バリュー資本の逆張り操作は価格の下支えを提供し、価格の一方的な下落確率を低下させ、順張り戦略の期待収益を減少させます。合理的なモメンタム資本はこれにより規模を縮小します。

**V → L：信頼の回復**

バリュー資本の参入は「市場には底がある」というシグナルを発し、マーケットメイカーの極端な損失への懸念を軽減し、流動性の回帰を促します。バリュー資本はマーケットメイカーの「保険」です。

**L → V：取引の便宜提供**

十分な流動性により、バリュー資本は低コストでポジションを構築でき、大口注文でも過度のスリッページを生じず、バリュー資本の資金効率を高めます。

## フィードバックループ

### 正のフィードバックループ（不安定）

$$M \uparrow \to \sigma \uparrow \to L \downarrow \to \text{価格衝撃} \uparrow \to \sigma \uparrow \to M \uparrow \text{ (または強制決済)}$$

モメンタム資本の増加 → ボラティリティ上昇 → 流動性撤退 → 価格衝撃の激化 → ボラティリティのさらなる上昇 → モメンタム資本のさらなる増加または強制決済

これは**崩壊スパイラル**のメカニズムです。

### 負のフィードバックループ（安定）

$$|S - S^*| \uparrow \to \sigma \uparrow \to V \uparrow \to |S - S^*| \downarrow \to \sigma \downarrow \to L \uparrow$$

価格が内在価値から乖離 → ボラティリティ上昇 → バリュー資本の参入 → 価格の回帰 → ボラティリティ低下 → 流動性の回復

具体的な経路：
- **急落後**：$S < S^*$ → V が買い → 価格回復
- **急騰後**：$S > S^*$ → V が売り → 価格調整

これは**回復メカニズム**です。

## システムの相転移

システムの状態は、**どちらのループが支配的か**によって決まります：

- **正のフィードバック > 負のフィードバック**：システムは崩壊に向かう
- **負のフィードバック > 正のフィードバック**：システムは安定に向かう
- **臨界点**：システムは相転移の境界にある

相転移はシステムの**創発的結果**であり、事前に設定された閾値ではありません。システムは正負のフィードバックの競合の中で自発的に相転移を生み出します。

## 三体の生態的地位

| 資本タイプ     | 生態学的役割   | システム安定性への影響 | 収益源            |
| -------------- | -------------- | ---------------------- | ----------------- |
| モメンタム資本 M | エネルギー注入者 | 不安定化               | ボラティリティ × 方向判断 |
| 流動性資本 L   | 緩衝材         | 安定化（受動的）       | 売買スプレッド    |
| バリュー資本 V   | 負帰還制御器   | 安定化（能動的）       | 価値回帰          |

**生態的均衡**：健全な市場には三者共存が必要です。

- モメンタム資本が不足：市場は停滞し、変動も取引機会もない
- 流動性資本が不足：取引コストが高く、市場効率が低い
- バリュー資本が不足：市場は脆弱で、崩壊しやすく、ゼロに帰するリスクすらある

## 従来の分類との関係

| 従来の分類     | 本質的帰属     | 説明                     |
| -------------- | -------------- | ------------------------ |
| 投機家         | モメンタム資本 M | 順張り、レバレッジ増幅   |
| 投資家         | バリュー資本 V   | 逆張り、価値判断         |
| マーケットメイカー | 流動性資本 L   | 両建て価格提示、スプレッド獲得 |
| トレンドトレーダー | モメンタム資本 M | モメンタム戦略           |
| アービトラージャー | バリュー資本 V   | 価格差の収斂             |
| パッシブ・インデックスファンド | 近似 L       | リバランス時に弱い負のフィードバックを生む |

注意：同一の参加者でも、時と場合によって異なる役割を果たすことがあります。分類の本質は**行動パターン**であり、属性ラベルではありません。

## 三体の類推

市場の三体は、天体力学における三体問題と深い類似性を持ちます。

**類推の鍵**：三体問題の本質は、**質量が近い三つの天体が相互作用する**ことです。互角であるがゆえに、いずれの天体もシステムを支配できず、カオス的な振る舞いが生じます。

市場においても、M、V、L の三種類の資本は同様に**互角**です：

- M が V と L を大きく上回る場合：市場は一方的に急騰急落した後、ゼロに帰する（バブル崩壊）
- V が M と L を大きく上回る場合：市場はほとんど変動しない（停滞）
- L が M と V を大きく上回る場合：価格は完全に外部情報によって決定される（完全効率的市場）

三者が互角である時のみ、市場は真の複雑なダイナミクスを示します。

**天体三体からの教訓**：

- 二体問題には解析解がある（楕円軌道）
- 三体問題には一般に解析解がなく、初期値に敏感
- 軌道は周期的、準周期的、またはカオス的になりうる

**市場三体からの推論**：

- **長期予測は不可能**：システムは初期値に敏感
- **短期的特性は予測可能**：トレンドとボラティリティ・クラスタリングは短期的現象
- **統計的規則性は頑健**：収益率分布のファットテール、ボラティリティ・クラスタリングなどの巨視的統計的特性は安定している

## 補足：資本規模の長期的進化

三体モデルの核心は M-V-L 間の相互作用ですが、前提となる問題があります：**なぜ三種類の資本は長期的に共存し、互角であり続けるのか？**

これは、**収益率駆動自然選択** (Return-Driven Selection) という従属メカニズムに依存しています。

### メカニズムの説明

同一の資本タイプ内において、個々の収益率は分布を示します。個体の行動（拡大、縮小、退出）はその収益率と強く相関します：

- 高収益の個体は残留または拡大する傾向
- 低収益の個体は縮小または退出する傾向

多数の個体の統計的効果により、この傾向性はその資本タイプの総量の変化として現れます。

### 自発的調整

ある資本タイプが過剰な場合：

1. 内部競争が激化
2. 平均収益率が低下
3. 限界的な個体の退出が増加
4. その資本タイプの総量が縮小

ある資本タイプが不足している場合：

1. 内部競争が緩和
2. 平均収益率が上昇
3. 新規資本の流入を誘引
4. その資本タイプの総量が拡大

### 各資本タイプの収益源と競争

| 資本タイプ | 収益源               | 過剰時の内部競争（内巻き）の現れ方           |
| ---------- | -------------------- | -------------------------------------------- |
| M          | ボラティリティ、トレンド継続 | 相互踏み上げ、スリッページ増大、トレンドの先食い |
| V          | 価値乖離、平均回帰   | 価値の窪地が奪い合われ、安全余裕が消失       |
| L          | 取引量、売買スプレッド | スプレッド縮小、マーケットメイキング利益の薄利化 |

### 理論的位置づけ

収益率駆動自然選択は**遅い変数**のメカニズム（週から年の時間スケール）であり、三体相互作用は**速い変数**のメカニズム（秒から日の時間スケール）です。

この従属メカニズムは、三体システムの**存在性**と**持続性**——なぜ市場がある一つの資本タイプが独占する状態に進化しないのか——を説明しますが、三体相互作用の核心的なダイナミクスは変えません。

## 研究方向

1.  **相空間構造**：アトラクター、リペラー、分離曲面
2.  **時間スケール分離**：速い変数（価格）、遅い変数（資本構造）
3.  **統計的特性**：エルゴード性、不変測度、滞在時間分布
4.  **力学方程式**：この枠組みに基づく SDE システム（別稿で詳述）
