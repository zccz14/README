# README

こんにちは、`CZ` または `zccz14` と申します。

専門分野は、**定量取引**と**フィンテック**です。

[No Trade No Life (NTNL)](https://github.com/No-Trade-No-Life/) の創設者です。

高エネルギー消費型の脳の持ち主で、思考と執筆を好み、物事が理にかなって自然に成就することを追求しています。

「朝に道を聞かば、夕べに死すとも可なり」。まだ死にたいとは思いませんし、世俗的な意味での成功も追求しています。それは、より多くのことを体験し、より深く理解する能力を得るためでもあります。人生はすべて体験のためであり、元々意味などないものです。すべてを理解することが、私にとっての意味です。

母語は中国語で、英語と日本語でのコミュニケーションが可能です。現在スペイン語を学習中であり、将来的には18か国語で交流できることを目指しています。異なる文化と言語を理解することは、世界を理解するための重要な道筋です。

このサイトは、個人的な思考、記録、見解を保管する場所として使用しています。

もし私がいなくなった日が来たら、これらはAIによって私を復活させるための原材料となります。
