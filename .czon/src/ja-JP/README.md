# README

こんにちは、こちらは `CZ`、または `zccz14` です。

専門分野は、量化取引とフィンテックです。

[No Trade No Life (NTNL)](https://github.com/No-Trade-No-Life/) の創設者です。

高エネルギー消費型の脳の持ち主で、思考と執筆を好み、物事が理にかなって自然に成就することを追求しています。

「朝に道を聞かば、夕べに死すとも可なり」。まだ死にたいとは思いませんし、世俗的な意味での成功も追い求めています。それは、より多くのことを体験し、より多くを理解する能力を得るためでもあります。人生はすべて体験のためであり、元々意味などありません。すべてを理解することが、私にとっての意味です。

母語は中国語で、英語と日本語でのコミュニケーションが可能です。現在スペイン語を学習中であり、将来的には18か国語で交流できることを望んでいます。異なる文化と言語を理解することは、世界を理解するための重要な道筋です。

このサイトは、個人的な考察、記録、見解を保管するための場所です。

もし私がいなくなった日が来たら、これらはAIが私を復活させるための原材料となるでしょう。
