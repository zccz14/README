# README

こんにちは、こちらは `CZ`、または `zccz14` です。

専門分野は、量化取引とフィンテックです。

[No Trade No Life (NTNL)](https://github.com/No-Trade-No-Life/) の創設者です。

高エネルギー消費型の脳の持ち主で、思考と執筆を好み、物事が理にかなって自然に成就することを追求しています。

「朝に道を聞かば、夕べに死すとも可なり」。まだ死にたいとは思いませんし、世俗的な意味での成功も追い求めています。それは、より多くを体験し、より多くを理解する能力を得るためでもあります。人生はすべて体験のためであり、元々意味などありません。すべてを理解することが、私にとっての意味です。

このサイトは、個人的な思考、記録、洞察をいくつか保管しておく場所です。

もしある日、私がいなくなったら、これらはAIが私を復活させるための原材料となります。
