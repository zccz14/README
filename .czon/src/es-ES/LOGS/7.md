2026-01-09 Memorándum de Funcionalidad

CZON puede generar múltiples tipos de resúmenes para cada artículo, para satisfacer diferentes usos y necesidades de plataformas. A continuación se presentan algunos tipos comunes de resúmenes y sus usos:

## Clasificación por Longitud/Formato

| Longitud/Formato | Nº de Caracteres | Ejemplo de Uso                              | Ejemplo de Prompt de Generación                           |
| ---------------- | ---------------- | ------------------------------------------- | --------------------------------------------------------- |
| **Micro-resumen**| 50-100 caracteres| Meta descripción SEO, descripción corta para tarjetas sociales | "Resume el valor central del artículo en una frase, no más de 100 caracteres" |
| **Resumen corto**| 150-250 caracteres| Resumen en páginas de listado de artículos, descripción para feed RSS | "Resume el contenido principal del artículo en 2-3 frases, destacando el punto central" |
| **Resumen medio**| 300-500 caracteres| Contenido para envío por email, introducción para newsletter | "Resume el artículo en un párrafo, incluyendo argumentos clave y conclusión" |
| **Resumen largo**| 800-1200 caracteres| Guion para locución, versión para navegación rápida | "Resume detalladamente la estructura completa del artículo, incluyendo argumentos principales y ejemplos" |
| **Resumen por puntos clave** | 5-8 puntos      | Escaneo rápido, esquema para presentaciones | "Extrae 5 puntos clave del artículo, listándolos con viñetas" |
| **Resumen de esquema** | Estructura jerárquica | Navegación de contenido, mapas mentales     | "Extrae el esquema de la estructura jerárquica del artículo, incluyendo títulos de todos los niveles" |

## Clasificación por Escenarios de Uso Específicos

### 1. **Optimización SEO**

- **Meta Description**: 50-160 caracteres, incluye palabras clave objetivo, atractivo para clics.
- **Resumen para datos estructurados**: Para marcado schema.org (FAQ, HowTo, Article, etc.).
- **Sugerencias de optimización de título**: Genera múltiples títulos alternativos (en forma de pregunta, con números, tipo guía, etc.).

### 2. **Distribución en Redes Sociales**

- **Twitter/X**: ~280 caracteres, puede necesitar dividirse en varios posts, con sugerencias de hashtags.
- **LinkedIn**: ~300 caracteres, tono más profesional y formal.
- **Facebook**: ~150 caracteres, tono más relajado y fácil de entender.
- **Instagram**: ~125 caracteres + sugerencias de emojis.
- **Threads**: 500 caracteres, tono conversacional.
- **Xiaohongshu (Pequeño Libro Rojo)**: ~100 caracteres, más coloquial, con etiquetas de tema.

### 3. **Vista Previa y Navegación de Contenido**

- **Resumen de índice**: Mostrado al inicio del artículo, "Este artículo cubrirá: 1... 2... 3...".
- **Indicador de progreso de lectura**: Breves avances para cada sección.
- **Resumen para emparejamiento de artículos relacionados**: Para tarjetas de recomendación del tipo "También te puede gustar".

### 4. **Accesibilidad y Funciones de Asistencia**

- **Resumen como texto alternativo (Alt-text)**: Genera descripciones detalladas para gráficos complejos e infografías.
- **Guion para podcast/audio**: Versión apta para lectura en voz alta, añade marcas para pausas naturales.
- **Versión simplificada**: Reduce la dificultad de comprensión para lectores principiantes.

### 5. **Gestión y Análisis de Contenido**

- **Resumen para edición**: Para editores/revisores, señala puntos destacados y áreas de mejora del artículo.
- **Resumen de palabras clave**: Extrae términos centrales, entidades nombradas, conceptos técnicos.
- **Análisis de sentimiento/tono**: Tono general del artículo (académico, humorístico, crítico, etc.).

### 6. **Usos Comerciales**

- **Resumen para comunicado de prensa**: Versión apta para distribución a medios.
- **Resumen para documentación de producto**: Puntos destacados de funcionalidades, problemas resueltos, usuarios objetivo.
- **Resumen para material de ventas**: Transformado en descripciones de puntos débiles del cliente y soluciones.

## Perspectivas Clave

Diferentes plataformas tienen diferentes "lenguajes": Twitter necesita un gancho, LinkedIn necesita profesionalidad, Xiaohongshu necesita cercanía.

El resumen también es una estrategia de contenido: Un buen resumen puede aumentar la tasa de clics en más de un 30%.

La automatización libera tiempo de creación: Los autores ya no tienen que reescribir manualmente el texto para cada plataforma.

Recomendación más práctica: Implementar primero los resúmenes SEO y de vista previa para listados, ya que son necesidades básicas para cualquier blog/sitio web. Los resúmenes para redes sociales pueden dejarse para una segunda fase, porque las preferencias de canales sociales varían mucho entre usuarios.

Con este diseño, CZON no solo ayuda a los usuarios a traducir contenido, sino también a empaquetarlo y distribuirlo, completando así un ciclo de valor más integral.
