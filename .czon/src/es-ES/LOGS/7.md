CZON puede generar múltiples tipos de resúmenes para cada artículo, para satisfacer diferentes usos y necesidades de plataformas. A continuación se presentan algunos tipos comunes de resúmenes y sus aplicaciones:

## Clasificación por Longitud/Formato

| Longitud       | Número de Caracteres | Ejemplo de Uso                              | Ejemplo de Prompt de Generación                           |
| -------------- | -------------------- | ------------------------------------------- | --------------------------------------------------------- |
| **Micro-resumen** | 50-100 caracteres    | Meta descripción SEO, descripción corta en tarjetas sociales | "Resume el valor central del artículo en una frase, no más de 100 caracteres" |
| **Resumen Corto** | 150-250 caracteres   | Resumen en listas de artículos, descripción para feed RSS    | "Resume el contenido principal del artículo en 2-3 frases, destacando el punto clave" |
| **Resumen Medio** | 300-500 caracteres   | Contenido para envío por email, introducción para newsletter | "Resume el artículo en un párrafo, incluyendo argumentos clave y conclusión" |
| **Resumen Largo** | 800-1200 caracteres  | Guion para locución, versión para lectura rápida             | "Resume detalladamente la estructura completa del artículo, incluyendo argumentos principales y ejemplos" |
| **Resumen por Puntos** | 5-8 puntos clave     | Escaneo rápido, esquema para presentaciones                  | "Extrae 5 puntos clave del artículo, listándolos con viñetas" |
| **Resumen de Esquema** | Estructura jerárquica | Navegación de contenido, mapas mentales                      | "Extrae el esquema de la estructura jerárquica del artículo, incluyendo títulos de todos los niveles" |

## Clasificación por Escenarios de Uso Específicos

### 1. **Optimización SEO**

- **Meta Description**: 50-160 caracteres, incluye palabras clave objetivo, atractiva para clics.
- **Resumen para Datos Estructurados**: Para marcado schema.org como FAQ, HowTo, Article, etc.
- **Sugerencias de Optimización de Títulos**: Genera múltiples títulos alternativos (preguntas, numéricos, guías, etc.).

### 2. **Distribución en Redes Sociales**

- **Twitter/X**: ~280 caracteres, puede necesitar dividirse en varios posts, con sugerencias de hashtags.
- **LinkedIn**: ~300 caracteres, tono más profesional y formal.
- **Facebook**: ~150 caracteres, tono más relajado y fácil de entender.
- **Instagram**: ~125 caracteres + sugerencias de emojis.
- **Threads**: 500 caracteres, tono conversacional.
- **Xiaohongshu (Pequeño Libro Rojo)**: ~100 caracteres, más coloquial, con etiquetas de tema.

### 3. **Vista Previa y Navegación de Contenido**

- **Resumen de Contenido**: Mostrado al inicio del artículo, "Este artículo cubrirá: 1... 2... 3...".
- **Indicador de Progreso de Lectura**: Breve avance de cada sección.
- **Resumen para Coincidencia de Artículos Relacionados**: Para tarjetas de recomendación "También te puede gustar".

### 4. **Accesibilidad y Funciones de Asistencia**

- **Resumen como Texto Alternativo (Alt-text)**: Genera descripciones detalladas para gráficos complejos e infografías.
- **Guion para Podcast/Audio**: Versión apta para lectura en voz alta, con marcas para pausas naturales.
- **Versión Simplificada**: Reduce la dificultad de comprensión para lectores principiantes.

### 5. **Gestión y Análisis de Contenido**

- **Resumen Editorial**: Para editores/revisores, señala puntos destacados y áreas de mejora del artículo.
- **Resumen de Palabras Clave**: Extrae términos centrales, entidades nombradas, conceptos técnicos.
- **Análisis de Sentimiento/Tono**: Tono general del artículo (académico, humorístico, crítico, etc.).

### 6. **Usos Comerciales**

- **Resumen para Comunicado de Prensa**: Versión apta para distribución a medios.
- **Resumen de Documentación de Producto**: Puntos destacados de funcionalidades, problemas resueltos, usuarios objetivo.
- **Resumen para Material de Ventas**: Transforma en descripciones de puntos débiles del cliente y soluciones.

## Perspectivas Clave

Diferentes plataformas tienen diferentes "lenguajes": Twitter necesita un gancho, LinkedIn necesita profesionalidad, Xiaohongshu necesita cercanía.

El resumen también es una estrategia de contenido: Un buen resumen puede aumentar la tasa de clics en más de un 30%.

La automatización libera tiempo de creación: Los autores ya no tienen que reescribir manualmente el texto para cada plataforma.

La recomendación más práctica: Implementar primero el resumen SEO y el resumen para vista previa en listas, ya que son necesidades básicas para cualquier blog/sitio web. Los resúmenes para redes sociales pueden dejarse para una segunda fase, ya que las preferencias de canales sociales varían mucho entre usuarios.

Con este diseño, CZON no solo ayuda a los usuarios a traducir contenido, sino también a empaquetarlo y distribuirlo, completando así el ciclo de valor de manera más integral.
