2026-01-09 Memorándum de Funcionalidad

CZON puede generar múltiples tipos de resúmenes para cada artículo, para satisfacer diferentes usos y necesidades de plataformas. A continuación se presentan algunos tipos comunes de resúmenes y sus usos:

## Clasificación por Longitud/Formato

| Longitud         | Caracteres       | Ejemplo de Uso                          | Ejemplo de Prompt de Generación                     |
| ---------------- | ---------------- | --------------------------------------- | --------------------------------------------------- |
| **Micro-resumen**   | 50-100 caracteres | Meta descripción SEO, descripción corta para tarjetas sociales | "Resume el valor central del artículo en una frase, no más de 100 caracteres" |
| **Resumen corto**   | 150-250 caracteres | Resumen en página de listado de artículos, descripción para feed RSS | "Resume el contenido principal del artículo en 2-3 frases, destacando los puntos clave" |
| **Resumen medio**   | 300-500 caracteres | Contenido para envío por correo, introducción para newsletter | "Resume el artículo en un párrafo, incluyendo argumentos clave y conclusión" |
| **Resumen largo**   | 800-1200 caracteres | Guion para locución de voz, versión para navegación rápida | "Resume detalladamente la estructura completa del artículo, incluyendo argumentos principales y ejemplos" |
| **Resumen de puntos clave** | 5-8 puntos clave | Escaneo rápido, esquema para presentaciones | "Extrae 5 puntos clave del artículo, listándolos con viñetas" |
| **Resumen esquemático** | Estructura jerárquica | Navegación de contenido, mapas mentales | "Extrae el esquema de la estructura jerárquica del artículo, incluyendo títulos de todos los niveles" |

## Clasificación por Escenarios de Uso Específicos

### 1. **Optimización SEO**

- **Meta Description**: 50-160 caracteres, incluye palabras clave objetivo, atrae clics
- **Resumen para datos estructurados**: Utilizado para marcado schema.org como FAQ, HowTo, Article, etc.
- **Sugerencias para optimización de títulos**: Genera múltiples títulos alternativos (en forma de pregunta, con números, estilo guía, etc.)

### 2. **Distribución en Redes Sociales**

- **Twitter/X**: ~280 caracteres, puede necesitar dividirse en múltiples publicaciones, con sugerencias de hashtags
- **LinkedIn**: ~300 caracteres, tono más profesional y formal
- **Facebook**: ~150 caracteres, tono más relajado y fácil de entender
- **Instagram**: ~125 caracteres + sugerencias de emojis
- **Threads**: 500 caracteres, tono conversacional
- **Xiaohongshu (Pequeño Libro Rojo)**: ~100 caracteres, más coloquial, con etiquetas de temas

### 3. **Vista Previa y Navegación de Contenido**

- **Resumen de tabla de contenidos**: Mostrado al inicio del artículo, "Este artículo cubrirá: 1... 2... 3..."
- **Indicador de progreso de lectura**: Breves avances para cada sección
- **Resumen para emparejamiento de artículos relacionados**: Utilizado para tarjetas de recomendación "También te puede gustar"

### 4. **Accesibilidad y Funciones de Asistencia**

- **Resumen para texto alternativo (Alt-text)**: Genera descripciones detalladas para gráficos complejos e infografías
- **Guion para podcast/audio**: Versión adecuada para lectura en voz alta, añade marcas de pausas naturales
- **Versión simplificada**: Reduce la dificultad de comprensión para lectores principiantes

### 5. **Gestión y Análisis de Contenido**

- **Resumen editorial**: Dirigido a editores/revisores, señala los puntos destacados y áreas de mejora del artículo
- **Resumen de palabras clave**: Extrae términos centrales, entidades nombradas, conceptos técnicos
- **Análisis de sentimiento/tono**: Tono general del artículo (académico, humorístico, crítico, etc.)

### 6. **Usos Comerciales**

- **Resumen para comunicado de prensa**: Versión adecuada para distribución a medios
- **Resumen para documentación de producto**: Puntos destacados de funcionalidades, problemas resueltos, usuarios objetivo
- **Resumen para material de ventas**: Transformado en descripciones de puntos débiles del cliente y soluciones
