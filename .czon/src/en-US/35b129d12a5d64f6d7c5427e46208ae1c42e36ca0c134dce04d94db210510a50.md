## 4. Quantitative Trading Strategies

### 4.1 EA Project Structure

| :--- | :----- | :--- | :----- |
| Senior Tranche | Priority in receiving profits and principal repayment | Low | Floating (currently ~12% annualized) |

### 4.2 Full Spectrum Analysis (FSA)

FSA is an investment decision framework proposed by the author. Its core elements include:

1.  **Outcome Space Design**: Deterministic returns, deterministic probabilities, mutual exclusivity, completeness.
2.  **Optimal Leverage Calculation**: Solves the compound return maximization problem using Newton's iteration method.
3.  **Black Swan Defense**: Artificially adds low-probability extreme events (beyond ±3σ).
4.  **Backtesting Method**: Gross Profit Margin (GPM) assesses model capacity.

### 4.3 Floating Profit Pyramid Strategy

Core Principles:
*   Add positions when profitable, cut losses when unprofitable.
*   Stop-loss condition: Loss reaches the level before adding positions.
*   Theoretical basis: Volatility clustering characteristics.
