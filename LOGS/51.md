## CZON 移除了 YAML Front Matter

CZON 发布了 0.8.6 版本。

主要是删除了 YAML Front Matter，目前通过 AI 提取 Metadata 的功能已经足够强大，不需要再手动添加这些信息了。

过去，CZON 会先组装 YAML FrontMatter，然后通过 AI 翻译成目标语言，但是**对于如果 YAML 中的内容格式比较复杂，例如包含引号，AI 翻译时往往无法保持格式正确**。

解决方案是直接让 AI 从 JSON to JSON 翻译 Metadata，JSON 格式是各个 AI 广泛训练过的，但 YAML 不是。

另外，出于**不修改原文但需要重新提取 Metadata 的场景**，分离 Metadata 和内容也是有好处的，它们会分离处理，在渲染阶段再合并成最终的 HTML。例如 CZON 未来会新增、实验性地支持一些新的 Metadata 字段，但是我不希望用户需要重新翻译整个文章，用户只需要翻译 Metadata 即可。

翻译 Metadata 的时候使用了逐语言逐文件的翻译，可能会增加一些 AI 调用的次数，但对于按量付费的 AI 服务来说，没有太大影响。

换句话说，**强烈不建议使用按次付费的 AI 服务**。顺便一提，我认为翻译功能可以使用免费级别的 AI 服务，因为这个任务很简单，几乎用不到什么推理能力。

另外，CZON 添加了更多的 Metadata 字段，并优化了一些 SEO 相关的内容。
